# 评论 IP属地与省份人口相关性仪表盘

一个本地 HTML 仪表盘，用于探索公开评论区中平台展示的 IP属地省份分布，与第七次全国人口普查分省常住人口分布之间的相关性和偏离。

## 内容

- `index.html`：可直接打开的交互式仪表盘。
- `collect_netease_stratified.mjs`：网易云音乐公开评论属地的分层抽样聚合脚本。
- `netease-stratified-sample.json`：当前默认样本的省份聚合结果和抽样诊断。

## 当前默认样本

- 平台：网易云音乐
- 抽样方式：9 个公开榜单分层抽样，每榜最多 24 首歌，每首最多 3 个评论时间窗口，每窗口 80 条。
- 原始公开评论：35,084
- 匹配大陆 31 省：34,439
- 排除海外、港澳台、未知等：645

## 隐私边界

只使用平台公开展示的 IP属地省份并立即聚合。仓库不保存真实 IP、用户名、主页、头像、评论原文或评论级记录。

## 本地运行

直接打开 `index.html`，或在目录中启动静态服务：

```bash
python -m http.server 8876
```

然后访问 `http://127.0.0.1:8876/index.html`。
